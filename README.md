# Kaubariškio bendruomenė

This repository contains the source code and technical files for the official non-commercial website of the Kaubariškis community in Panevėžys District, Lithuania.

## Project purpose

The website is developed and maintained to provide local residents and visitors with accessible information about:

- community news;
- upcoming events and activities;
- community projects and initiatives;
- public announcements;
- documents and forms;
- volunteer opportunities;
- event registration;
- photographs and reports from community activities;
- contact information.

The website is not an online store and does not process payments or payment-card information.

## Repository purpose

This GitHub repository is used for legitimate website-development and maintenance activities, including:

- storing and updating source code;
- maintaining version history;
- testing website changes;
- managing website structure and design;
- maintaining public forms and documents;
- preserving project files and technical documentation;
- publishing the community website.

## Technologies

Depending on the current project version, the repository may contain:

- HTML;
- CSS;
- JavaScript;
- TypeScript;
- React components;
- configuration files;
- images and public documents;
- forms and registration tools;
- other website-development assets.

## Data protection

Private personal information must not be stored directly in this public repository.

The repository should not contain:

- passwords;
- secret API keys;
- access tokens;
- payment information;
- private membership records;
- confidential personal documents.

Public forms and registration tools should collect only the information required for the stated community purpose.

## Project status

The Kaubariškis community website is an active community project and may be updated as community needs, events and activities change.

Some website sections or functions may still be under development.

## Apie projektą lietuviškai

Šioje saugykloje laikomas oficialios nekomercinės Kaubariškio bendruomenės interneto svetainės programinis kodas ir techniniai failai.

Svetainėje skelbiama:

- bendruomenės informacija ir naujienos;
- renginiai ir veiklos;
- bendruomenės projektai;
- vieši pranešimai;
- dokumentai ir formos;
- registracijos į renginius;
- veiklų nuotraukos ir aprašymai;
- kontaktinė informacija.

Ši „GitHub“ saugykla naudojama svetainės programiniam kodui saugoti, pakeitimams registruoti, svetainės funkcijoms testuoti ir projektui prižiūrėti.
